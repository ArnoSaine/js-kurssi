* Monipuolinen kirjasto sovelluksen kääntämiseen eri kielille

```js
<FormattedMessage
  id=""
  values={{
    name: 'Eric'
  }}
/>
```

Käännöstekstien tiedostot, kuten `messages.fi-FI.properties`:
```
app.greeting = Hello, {name}!
```
