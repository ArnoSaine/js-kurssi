* Monipuolinen kirjasto sovelluksen kääntämiseen eri kielille

```js
<FormattedMessage
  id="app.greeting"
  values={{
    name: 'Eric'
  }}
/>
```

Käännöstiedostot, kuten `messages.fi-FI.properties`:
```
app.greeting = Hei, {name}!
```
