# Objektimäärityksen lyhenne

```javascript
const henkilö = {
  id,
  nimi
};

// Sama vanhalla syntaksilla:
const henkilö = {
  id: id,
  nimi: nimi
};
```

* Paikallinen muuttuja tulee nimettyä järkevästi, mikäli sen arvo halutaan jossain vaiheessa asettaa objektille propertynä
